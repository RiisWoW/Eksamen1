Server

En server er navnet vi bruger for en computer, som deler sin data. Dette er ofte en computer, med meget stor lager kapacitet, hvor andre enheder(klienter/clients) herved kan hente dataen.

Forbindelse til server

Man forbinder til en server via netværket. Når det kommer til en hjemmeside, forbinder vi via en browser. Dette gør vi ved hjælp af et domænenavn. For at vi kan få domænenavnet til at være brugbart, skal vi bruge en DNS

DNS

DNS eller Domain name system, er et system, som forbinder dit domænenavn med en IPv4(255.255.255.255) eller IPv6(2001:0DB8:AC10:FE01::) adresse.

FTP

FTP eller File transfer protocol, bruges til at forbinde og flytte indhold til serveren, fra din lokale enhed. Dette gøres igennem en FTP-client, som winSCP eller FileZila.

Kode

Kode er essencen af vores hjemmesider, her bruger vi HTML, CSS & JS.

HTML

HTML eller Hypertext Markup Language, er det sprog vi bruger, når vi skal lave indholdet på en hjemmeside, og fortælle en browser, hvordan indholdet er markeret.

For en bedre forståelse af hvad HTML er, skal vi kigge på hvad Hypertext er.

Hypertext er måden vi forbinder internettet, igennem links.

Tags

HTML Tags, laves ved at vælge tagget man vil bruge.

I dette eksempel bruger vi tagget heading1 eller h1, dette gør vi ved at bruge følgende <h1>.

Du skal herefter afslutte dit tag, ved at skrive følgende følgende </h1> . Her i mellem vil vi indsætte den tekst vi vil have som heading1.

Dette gøres som følgende <h1>Hello World</h1>

Semantiske Tags

Semantiske tags, er tags i HTML med betydning, denne betydning bruges af f.Eks. søge maskiner, for at se indelingen af siden. Rigtig brug af semantiske tags, kan hjælpe din SEO.

De forskellige semantiske tags, har forskellige betydninger.

Her er nogen af de forskellige semantiske tags

  • </header>
  • </footer>
  • </nav>
  • </section>
Attributes

Attributes, er yderligere information, som man tilgiver et tag.

Dette kan være en class eller et ID.

Disse bruges som selector i CSS, og gør så du kan ramme specifikke eller en gruppe tags.

Eksempel på HTML
CSS

CSS eller Cascading style sheets, er det kode sprog vi bruger, når vi designer vores hjemmeside.

Det gøres ved at bruge selectors, og derved tildele nogen beskrivelser til disse selectors, som beskriver hvordan HTML elementerne skal vises på forskellige enheder.

CSS går også ind og fjerne behovet for at style sin HTML igennem html attributes

Display funktioner

I CSS har vi nogen forskellige Display funktioner, som vi bruger til at i fortælle hvordan og om et HTML element skal vises.

Her er nogen af de forskellige display egenskaber

  • Block
  • Grid
  • Flex
  • Table

En af de vigtigste display funktioner, vil være Grid. Det er fordi at det giver dig muligheden for at fuld kontrol over opsættelsen af dine elementer. Derudover giver det dig muligheden, hvis du laver et Grid-template-area , at lave hurtige ændringer i hvor elementer skal være, i forhold til hinnanden.

Responsivt design

Responsivt design, betyder at vi laver et design, som tilpasser sig til den måde brugeren bruger siden. F.Eks. Designer så der er et design som fungere på mobil enheder, da et traditonelt desktop design ikke virker der.

Dette kan gøres ved at bruge en @media . Hvor vi så kan gå ind og specificere, ud fra hvilket media, siden skal ændre sig.

Her er et eksempel på hvordan @media screen and (min-width: 1080px) Dette betyder at de ændringer du laver under denne Media, bliver set hvis skærmen er over 1080px bred.
