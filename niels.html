1. semester

Server

En server er navnet vi bruger for en computer, som deler sin data. Dette er ofte en computer, med meget stor lager kapacitet, hvor andre enheder(klienter/clients) herved kan hente dataen.

Forbindelse til server

Man forbinder til en server via netværket. Når det kommer til en hjemmeside, forbinder vi via en browser. Dette gør vi ved hjælp af et domænenavn. For at vi kan få domænenavnet til at være brugbart, skal vi bruge en DNS

DNS

DNS eller Domain name system, er et system, som forbinder dit domænenavn med en IPv4(255.255.255.255) eller IPv6(2001:0DB8:AC10:FE01::) adresse.

FTP

FTP eller File transfer protocol, bruges til at forbinde og flytte indhold til serveren, fra din lokale enhed. Dette gøres igennem en FTP-client, som winSCP eller FileZila.
